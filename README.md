## 간단한 컨벤션 지침

### 클래스
* UpperCamelCase를 사용합니다.
* 익히 알려진 경우 (ex. DTO, URL 등) 를 제외하고는 합의되지 않은 약어를 사용하지 않도록 합니다.

### 인터페이스
* UpperCamelCase를 사용합니다.
* 인터페이스명에 직접적으로 인터페이스임을 명시하지 않습니다.

### 메소드
* lowerCamelCase를 사용합니다.
* 메소드가 어떤 역할을 하는지 최대한 문장형으로 작명 부탁드립니다!

### 변수
* lowerCamelCase를 사용합니다.

### 상수
* UPPER_SNAKE_CASE를 사용합니다.
